Vietnamese Input Test - Telex Method

Telex Input Method

The Vietnamese input system now uses only the Telex method for typing Vietnamese characters with diacritics.

Telex Input Examples:

aa → â (circumflex)
aw → ă (breve)
ee → ê (circumflex)
oo → ô (circumflex)
ow → ơ (horn)
uw → ư (horn)
dd → đ (stroke)

s → sắc (acute) - á, ấ, ắ, é, ế, í, ó, ố, ớ, ú, ứ, ý
f → huyền (grave) - à, ầ, ằ, è, ề, ì, ò, ồ, ờ, ù, ừ, ỳ
r → hỏi (hook) - ả, ẩ, ẳ, ẻ, ể, ỉ, ỏ, ổ, ở, ủ, ử, ỷ
x → ngã (tilde) - ã, ẫ, ẵ, ẽ, ễ, ĩ, õ, ỗ, ỡ, ũ, ữ, ỹ
j → nặng (dot) - ạ, ậ, ặ, ẹ, ệ, ị, ọ, ộ, ợ, ụ, ự, ỵ
✅ VNI method has been removed. Only Telex method is now supported.

How to Test

  1. Open the main index.html file
  2. Go to Settings and enable "Vietnamese Input"
  3. Try typing Vietnamese characters using Telex method:
    • Type aa to get â
    • Type aas to get ấ
    • Type dd to get đ
    • Type ow to get ơ
